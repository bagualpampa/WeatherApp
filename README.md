# WeatherApp

## System Requirements:

- Visual Studio 2017
- IIS Express

## How to run it?

You can either start the app from VS itself or you can install it on IIS un run it from the browser.

**IIS Express command:**

```sh
iisexpress /path:d:\weatherapi\ /port:9090 /clr:v4.0
```

### API Help

```sh
/swagger
```
